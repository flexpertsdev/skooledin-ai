Implementation Guide

Dark Mode CSS Implementation

:root {
  /* Dark Mode Colors - Based on #14191C */
  --bg-primary: #1F2428;      /* Cards, panels */
  --bg-secondary: #14191C;    /* Base color - main background */
  --bg-tertiary: #2D3339;     /* Elevated elements */
  --bg-quaternary: #3A4046;   /* Higher elevation */
  
  --text-primary: #F7F9FA;    /* High contrast text */
  --text-secondary: #D1D9E0;  /* Medium contrast text */
  --text-tertiary: #8B949E;   /* Low contrast text */
  --text-muted: #6E7681;      /* Very subtle text */
  
  --primary-500: #16C181;     /* Main brand color */
  --primary-600: #10A574;     /* Hover states */
  --primary-700: #0F7860;     /* Pressed states */
}

body {
  background-color: var(--bg-secondary);
  color: var(--text-primary);
}

.card {
  background: var(--bg-primary);
  border: 1px solid var(--border-primary);
}

Usage Example

/* Direct usage of dark mode variables */
.my-component {
  background: var(--bg-primary);
  color: var(--text-primary);
  border: 1px solid var(--border-primary);
}

.my-button {
  background: var(--primary-500);
  color: white;
}

.my-button:hover {
  background: var(--primary-600);
}
